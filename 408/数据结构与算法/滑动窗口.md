### 题型1：

解决问题：两个字符串，从字符串1中寻找子串，使其等于字符串2的...

~~~java
public String minWindow(String s, String t) {
    // 用于记录需要的字符和窗口中的字符及其出现的次数
    Map<Character, Integer> need = new HashMap<>();
    Map<Character, Integer> window = new HashMap<>();

    int left = 0, right = 0;
    int valid = 0; // 窗口中满足需要的字符个数
	
    //1.扩大窗口
    while (right < s.length()) {
        // c 是将移入窗口的字符
        char r = s.charAt(right);
        // 扩大窗口
        right++;
        // 进行窗口内数据的一系列更新
        if (need.containsKey(r)) {
            window.put(r, window.getOrDefault(r, 0) + 1);
            if (window.get(r).equals(need.get(r)))
                valid++; // 只有当 window[r] 和 need[r] 对应的出现次数一致时，才能满足条件，valid 才能 +1
        }

        // 2.判断左侧窗口是否要收缩
        while (valid == need.size()) {
            // l 是将移出窗口的字符
            char l = s.charAt(left);
            // 缩小窗口
            left++;
            // 进行窗口内数据的一系列更新
            if (need.containsKey(l)) {
                //注意这里if与window.put的顺序与上面不同
                if (window.get(l).equals(need.get(l)))
                    valid--; // 只有当 window[l] 内的出现次数和 need[l] 相等时，才能 -1
                window.put(l, window.get(l) - 1);
            }
        }
    }
	
    return ...;
}
~~~

#### 求 等于目标值且长度最小的子数组

https://leetcode.cn/problems/minimum-size-subarray-sum/description/

心得：右指针一直移动，使用一个变量记录（如sum）右指针移动产生的结果,当满足条件时，记录满足条件产生的结果，左指针开始移动，直到条件再次变为不满足，右指针继续移动，满足条件、记录、左指针破坏条件......

解法1：my：

~~~java
class Solution {
    public int minSubArrayLen(int target, int[] nums) {
        int left = 0,right = 0;
        int n = nums.length;
        int sum = 0;
        int len = Integer.MAX_VALUE;
        while(right < n){
            int rightNum = nums[right];
            right++;
            sum += rightNum;
            while(sum >= target){
                len = Math.min(len,right - left);
                sum -= nums[left];
                left++;
            }
        }
        return len == Integer.MAX_VALUE ? 0 : len;
    }
}
~~~

~~~java
class Solution {
    public String minWindow(String s, String t) {
        
        Map<Character,Integer> window = new HashMap<>();
        Map<Character,Integer> need = new HashMap<>();
        int valid = 0;
        //左闭右开
        int left = 0,right = 0;
        int start = 0,len = Integer.MAX_VALUE;
        for(char c: t.toCharArray()){
            need.put(c,need.getOrDefault(c,0)+1);
        }
        while(right < s.length()){
            char r = s.charAt(right);
            right++;
            //需要的字符串才添加进窗口中
            if(need.containsKey(r)){
                window.put(r,window.getOrDefault(r,0)+1);
                if(window.get(r).equals(need.get(r))){
                    valid++;
                }
            }

            while(valid == need.size()){
                if(right - left < len){
                    start = left;
                    len = right-left;
                }
                char l = s.charAt(left);
                left++;
                if(need.containsKey(l)){
                    if(window.get(l).equals(need.get(l))){
                        valid--;
                    }
                    window.put(l,window.get(l)-1);
                }
            }

        }
        return len==Integer.MAX_VALUE?"":s.substring(start,start+len);
    }
}
~~~

~~~java
class Solution {
    public String minWindow(String s, String t) {
        if(s.length() < t.length()){
            return "";
        }
        // 记录每个字符与出现的数量
        Map<Character,Integer> need = new HashMap<>();
        Map<Character,Integer> window = new HashMap<>();
        int left = 0,right = 0,count = 0,start = 0;
        // int len = Integer.MAX_VALUE;
        int end = Integer.MAX_VALUE; // 区别
        // 记录每个字符的数量
        for(Character c : t.toCharArray()){
            need.put(c,need.getOrDefault(c,0) + 1);
        }
        while(right < s.length()){
            char r = s.charAt(right);
            right++;
            if(need.containsKey(r)){
                // 记录每个字符的数量
                window.put(r,window.getOrDefault(r,0) + 1);
                // 添加后检查是否有相同数量的字符
                if(window.get(r).equals(need.get(r))){
                    count++;
                }
            }
            // need与window全部的字符数量都相等,满足条件时
            while(count == need.size()){
                //记录
                if(right - left < end - start){
                    start = left;
                    end = right;
                }
                // 破坏条件
                char l = s.charAt(left);
                left++;
                if(need.containsKey(l)){
                    if(need.get(l).equals(window.get(l))){
                        count--;
                    }
                    window.put(l,window.get(l) - 1);
                }
            }
        }
        return end == Integer.MAX_VALUE ? "" : s.substring(start,end);
    }
}
~~~

~~~java
class Solution {
    public int dismantlingAction(String arr) {
        if(arr.length() == 0){
            return 0;
        }else if(arr.length() == 1){
            return 1;
        }
        Set<Character> window = new HashSet<>();
        int left = 0,right = 0,len = 0;
        while(right < arr.length()){
            char r = arr.charAt(right);
            right++;
            if(!window.contains(r)){
                window.add(r);
                // 还没有满足条件就结束的 "aab" 
                len = Math.max(len,right - left);
            }else{// 满足条件时
                while(window.contains(r)){
                    // 记录
                    len = Math.max(len,right - left - 1);
                    // 破坏条件
                    char l = arr.charAt(left);
                    left++;
                    window.remove(l);
                }
                window.add(r);
            }
        }
        return len;
    }
}
~~~

~~~java
        }
~~~

~~~java
class Solution {
    public boolean checkInclusion(String s1, String s2) {
        if(s1.length() > s2.length()){
            return false;
        }
        if(s1.length() == 1 && s1.equals(s2)){
            return true;
        }
        Map<Character,Integer> window = new HashMap<>();
        Map<Character,Integer> need = new HashMap<>();
        int left = 0,right = 0,count = 0,len = 0;
        for(char c : s1.toCharArray()){
            need.put(c,need.getOrDefault(c,0) + 1);
        }
        while(right < s2.length()){
            char r = s2.charAt(right);
            right++;
            if(need.containsKey(r)){
                window.put(r,window.getOrDefault(r,0) + 1);
                if(need.get(r).equals(window.get(r))){
                    count++;
                }
            }
            while(count == need.size()){
                len = right - left;
                if(len == s1.length()){ // 窗口里面包含s1的所有元素并且数量相等时返回true
                    return true;
                }
                char l = s2.charAt(left);
                left++;
                if(need.containsKey(l)){
                    if(window.get(l).equals(need.get(l))){
                        count--;
                    }
                    window.put(l,window.get(l) - 1);
                }
            }
        }
        return false;
    }
}
~~~

~~~java
class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        List<Integer> list = new ArrayList<>();
        if(s.length() == 1 && p.length() == 1 && s.equals(p)){
            list.add(0);
            return list;
        }
        // 长度问题时 直接返回空列表
        if(p.length() > s.length()){
            return list;
        }
        Map<Character,Integer> need = new HashMap<>();
        Map<Character,Integer> window = new HashMap<>();
        int left = 0,right = 0,count = 0;
        for(char c : p.toCharArray()){
            need.put(c,need.getOrDefault(c,0) + 1);
        }
        while(right < s.length()){
            char r = s.charAt(right);
            right++;
            if(need.containsKey(r)){
                window.put(r,window.getOrDefault(r,0) + 1);
                if(need.get(r).equals(window.get(r))){
                    count++;
                }
            }
            // 使用长度与count变量避免了对p字符串进行组合，只要left到right的窗口中有所需要的所有元素并且长度相同，那么肯定是p字符串组合的一种

                // 记录
                if(right - left == p.length()){
                    list.add(left);
                }
                // 破坏条件
                char l = s.charAt(left);
                left++;
                if(need.containsKey(l)){
                    if(need.get(l).equals(window.get(l))){
                        count--;
                    }
                    window.put(l,window.get(l) - 1);
                }
            }
        }
        return list;
    }
}
~~~
